Crystal Website
===============
Powered by [Jekyll](https://jekyllrb.com/)

## Development setup

- Checkout the repositorie
- Run `bundle install`
- Start the server with `bundle exec jekyll serve`
- Open a broswer in `localhost:4000`

or to build without starting it: `bundle exec jekyll build`
